DCLM Webcast

a fullstack app serving thousands weekly.

A simple but useful fullstack web app written to serve the needs of online viewers and members of the Deeper Life Bible Church, a global church headquartered in Lagos, Nigeria.

Web app screenshots
A simple web app with a far-reaching impact.

This simple web app servers over 10k weekly users and visitors with the live streaming messages of Pastor Dr. W.F. Kumuyi. It is such an honour to be privileged to work under God’s servant and this church Deeper Christian Life Ministry.

App tech stack: PHP, MySQL, EC2, AWS S3, AWS RDS, HTML, CSS and Javascript.
